Структура проэкта Магазин
/admin
 /assets  -- здесь  находиться файлы css , js, fonts для админки
 /img -- графические файлы для админки
 /modules - Модулю для админки
  - add-category.php - функционал для обавления категориями
  - add-product.php - функционал для добавления товара
  - change-status.php - меняем статут заказа
  - delete-category- функционал для удаления категориями
  - edit-category - функционал для редактирования категориям
  - edit-product - функционал для редактрования продукта
 /parts
  - footer.php - отдельно подключаеться футер
  - header.php  - отдельно подключаеться шапка
  - nav.php - отдельно подключаем левій сайдбар в админке

- categories.php - страничка для вывода категорий магазина, где можна изменять, добавлять, удалять
- index.php  - главная страница админки
- users.php - пользователи админки
- orders.php - страничка с заказами пользователя


/shop-master
  /admin - админка сайта 
  /assets  -- здесь  находиться файлы css , js, fonts для магазина. Где main.js - главный файл скриптов
  /img -- графические файлы для админки
  /configs 
    - db.php - файл подключения БД
  /modules  Модули магазина
    /basket - функционал для корзины
      - add-basket.php - функционал добавления товара в корзину
      - delete.php - удаление товара с корзины
      - edit-count.php - редактирование количества товаров в корзине
      - order.php - оформление товара
    /products - функционал связаный с продуктами 
      - get-more.php - функционал добавления на страницу товаров с помщью кнопки "Показать еще"
      - pag.php - функционал вывода и изминения пагинации на главной страничке
  /parts - отдельные подключаемые части сайта
   - cat_nav.php - левый сайдбар для вывода товаров по категориями
   - footer.php - отдельно подключаеться футер
   - header.php  - отдельно подключаеться шапка
   - product_cart.php  - отдельно подключаеться карточка товара
 - about.php - страничка в разроботке
 - basket.php - страничка корзины
 - cat.php - страничка с функционалом для вывода категорий
 - contacts.php - страничка в розработке
 - index.php  - главная страничка магазина
 - product.php  - страничка для отдельного вывода выбраного товара
 - search.php - страничка в розработке
 - login.php - страничка авторизации
 - registr.pho - страничка регистрации
 - bot.php - Телеграм бот
